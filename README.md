# ШИЗИ САУНД (Next.js)

## Запуск
npm install
npm run dev
Открой http://localhost:3000

## Yclients виджет
Скрипт подключен в pages/_app.tsx.
Все кнопки с действием бронирования открывают попап через window.yclients.open().

## Telegram (форма заявки)
Создай .env.local по примеру .env.example и перезапусти dev-сервер.
